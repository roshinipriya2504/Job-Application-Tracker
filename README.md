# Job Application Tracker (Java)

A console-based Java application to manage and track job/internship applications efficiently.

## Features
- Add new applications
- View all applications
- Search by company name
- Update application status
- Delete applications
- Generate HTML report for browser view

## Tech Stack
- Java
- ArrayList
- File Handling

## How to Run
1. Compile: javac *.java
2. Run: java Main
3. Choose option 6 to generate HTML report

## Output
Generates an `output.html` file to display applications in a browser.
